#### Задание 1.

Напишите блок схему отправки ping с устройства на устройство.

**Условия**: В блок схеме должно быть: 
- блоки ввода данных для ip
- как минимум один цикл
- блок вывода данных
```mermaid
flowchart TD

A([Start])-->B[Ввести IP-адрес назначения]

B-->C{IP корректен?}

C--Нет-->B

C--Да-->D[Отправить ICMP Echo Request]

D-->E[Ожидать ответ]

E-->F{Ответ получен?}

F--Нет-->G{Превышен таймаут?}

G--Нет-->E

G--Да-->H[Вывод: 'Request timed out']

F--Да-->I[Вывод: 'Reply from IP']

I-->J{Отправить ещё пакет?}

J-- Да-->D

J--Нет-->K([End])

H-->J
```
#### Задание 2.

Вспомните как работает протокол ospf, а в частности его алгоритм поиска кратчайшего пути. Составьте блок-схему отправки сообщения с одного устройства до другого, которые объединены несколькими маршрутизаторами с ospf протоколом.

**Условия**: В блок схеме должно быть: 
- должно быть как минимум одно условие
- должен быть как минимум один цикл
- должна быть как минимум одна определённая переменная
```mermaid
flowchart TD

A([Запуск OSPF процесса])-->B[Инициализация:Cost= бесконечности для всех узлов,кроме своего Cost=0]

B-->C[Построение LSDB через рассылу Hello-пакетов]

C-->D[Выбор узлас минимальным Cost из непосещенных]

D-->E{Все узлы обработаны?}

E--Да--> F([Завершение алгоритма])

E--Нет-->G[Помеитить узел как посещенный и обновить таблицу маршрутизации]

G-->H[Для каждого соседа выбранного узла: рассчитать новый Cost]

H-->I{Новый Cost < текущего?}

I--Да-->J[ОбновитьCost и путь]

I--Нет-->K[Перейти к следующему соседу]

J-->K

K-->L{Все соседи обработаны?}

L--Нет-->H

L--Да-->D

```
#### Задание 3.

**Условия**: 
- Должно быть условие проверки пароля
- У пользователя есть 3 попытки ввода пароля
- Должны быть блоки ввода и вывода информации
```mermaid 
flowchart TD
A["Начало"] --> B["Клиент: Подключение к SSH-серверу"]

D --> E["Клиент: Ввод логина и пароля"]
E --> F["Сервер: Проверка пароля"]
F --> G{"Пароль верный?"}
G -- Да --> H["Аутентификация успешна"]
H --> I["Сервер: Открытие сессии"]
I --> J["Работа в SSH-сессии"]
J --> K["Конец"]
G -- Нет --> L["Попытки = Попытки-1"]
L --> M{"Попытки>0?"}
```
