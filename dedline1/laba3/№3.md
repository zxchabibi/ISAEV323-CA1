### Практическая часть

#### Задание 1.

Напишите Sequence-диаграмму отправки ping-запроса на сервер.

**Условия**: В диаграмме должно быть:
- Запрос и ответ от сервера
```mermaid
sequenceDiagram

  

    participant Client

  

    participant Server

  

  

    Client->>Server: Ping Request

  

    activate Server

  

    Server-->>Client: Ping Response

  

    deactivate Server
```
#### Задание 2.

В теоретической части рассматривался пример авторизации на сайте. Напишите подобную sequence-диаграмму авторизации на сайте, но с двойной аутентификацией.

**Условия**: В диаграмме должно быть:
- Ввод пароля
- Двойная аутентификация (например по смс)
```mermaid
sequenceDiagram

    participant User

    participant WebBrowser

    participant WebServer

    participant AuthenticationServer

    participant SMSGateway

    User->>WebBrowser: Вводит логин и пароль

    WebBrowser->>WebServer: Отправляет логин и пароль

    WebServer->>AuthenticationServer: Запрос на аутентификацию с логином и паролем

    activate AuthenticationServer

    AuthenticationServer-->>WebServer: Подтверждение логина и пароля (Успешно или Ошибка)

    alt Успешная аутентификация логина и пароля

        AuthenticationServer->>SMSGateway: Запрос на отправку SMS с кодом подтверждения

        activate SMSGateway

        SMSGateway-->>User: Отправка SMS с кодом подтверждения

        deactivate SMSGateway

        User->>WebBrowser: Вводит код подтверждения из SMS

        WebBrowser->>WebServer: Отправляет код подтверждения

        WebServer->>AuthenticationServer: Запрос на подтверждение кода из SMS

        AuthenticationServer->>AuthenticationServer: Проверка кода подтверждения

        AuthenticationServer-->>WebServer: Подтверждение кода (Успешно или Ошибка)

        alt Успешное подтверждение кода

            AuthenticationServer->>WebServer: Аутентификация завершена успешно

            WebServer->>WebBrowser: Перенаправление на защищенную страницу

            WebBrowser->>User: Отображение защищенной страницы

        else Ошибка подтверждения кода

            AuthenticationServer->>WebServer: Аутентификация не удалась

            WebServer->>WebBrowser: Отображение сообщения об ошибке аутентификации

            WebBrowser->>User: Отображение сообщения об ошибке аутентификации

        end

    else Ошибка аутентификации логина и пароля

        AuthenticationServer-->>WebServer: Аутентификация не удалась

        WebServer->>WebBrowser: Отображение сообщения об ошибке аутентификации

        WebBrowser->>User: Отображение сообщения об ошибке аутентификации

    end

    deactivate AuthenticationServer
```
#### Задание 3.

Представьте что вы кассир в Пятерочке. К вам приходит покупатель с покупками. Визуализируйте вашу работу с покупателем и кассой.

**Условия**: 
- Должно быть как минимум 4 объекта
```mermaid
sequenceDiagram

    participant Покупатель

    participant Кассир

    participant Касса

    participant База данных

 

    Покупатель->>Кассир: Товары для сканирования

    activate Кассир

    loop Для каждого товара

        Кассир->>Касса: Сканирует товар

        activate Касса

        Касса->>База данных: Запрос цены

        База данных-->>Касса: Цена товара

        deactivate Касса

    end

    Кассир->>Касса: Завершает сканирование

    Касса->>Кассир: Отображает сумму

    Кассир->>Покупатель: Сообщает сумму

    Покупатель->>Кассир: Оплата

    Кассир->>Касса: Печать чека

    Касса-->>Покупатель: Выдает чек и товары

    deactivate Кассир
```
