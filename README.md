Простая утилита трассировки
Этот проект помогает:

1. Диагностировать сетевые проблемы (показывает путь до хоста).
2. Сохранять результаты в файл (просто или через модуль logging).
3. Надежно обрабатывать ошибки, например, если команда traceroute не найдена.

Как использовать:

1.  Сохраните код в файл, например, my_traceroute.py.
2.  Запустите его из командной строки:
    •   python my_traceroute.py (затем введите адрес по запросу)
    •   python my_traceroute.py google.com (адрес передается как аргумент)
    •   python my_traceroute.py 8.8.8.8
